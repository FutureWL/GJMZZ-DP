# 06 岗位 JD · 实施组 / 职能组 / 法务

> 本文档定义 **实施组(实施工程师、客户成功)** 和 **职能组(HR、行政、法务、财务)** 的岗位 JD。
> 配套:[01 总览](./01_团队组建总览.md) · [02 岗位体系](./02_组织架构与岗位体系.md) · [05 业务组 JD](./05_岗位JD_制造业务.md)

## 实施组

> **实施组 = 客户的脸面**。B2B 软件卖出去只是开始,真正决定续约率的是"实施能不能落地"。

---

## 🔴 P0-01:实施工程师

**【岗位职责】**

1. **项目实施**:
   - 主导客户现场实施(需求调研 → 方案设计 → 系统配置 → 数据准备 → 培训 → 试运行 → 上线);
   - 编写实施方案(部署文档、配置文档、培训手册);
   - 主导 UAT 验收(与制造业务架构师协作)。

2. **客户沟通**:
   - 日常对接客户方项目经理(PMO / IT 部门);
   - 协调客户方业务部门(总经理 / 制造副总 / 车间主任);
   - 协调公司内部(产品 / 技术 / 业务架构师)。

3. **系统配置**:
   - 平台基础配置(组织/工厂/产线/部门主数据);
   - 业务配置(角色/权限/审批流);
   - 报表配置(驾驶舱/看板)。

4. **培训**:
   - 客户方管理员培训(管理员手册);
   - 客户方最终用户培训(用户手册);
   - 客户方 IT 培训(运维手册)。

5. **上线支持**:
   - 上线陪跑(关键时期 7×12 待命);
   - 收集反馈,持续优化;
   - 编写"实施经验库"。

**【任职要求】**

- **学历**:本科及以上;
- **经验**:3 年以上 B2B 软件实施经验(ERP/MES/QMS/WMS 任一);
- **能力**:
  - 优秀的客户沟通能力(尤其面对"不懂 IT 的车间主任");
  - 良好的项目管理能力(能推动客户方按计划走);
  - 良好的文档能力(实施方案/培训手册);
  - 抗压能力(出差、加班、应对客户抱怨);
  - 基本的 SQL 能力(能跑数据);
- **行业经验**:有制造业 IT 实施经验(用友/金蝶/鼎捷/宝信出身加分);
- **加分**:
  - 有 PMP 认证;
  - 有 ISO 9001 内审员证;
  - 懂 BPMN 基础。

**【薪酬范围】** ¥10K-18K/月(13 薪)+ 差旅补贴 + 出差

---

## 🟡 P1-01:客户成功经理(CSM)

**【岗位职责】**

1. **客户关系**:
   - 高价值客户的 1v1 服务(QBR / 季度业务回顾);
   - 日常对接客户方高层(总经理 / 制造副总);
   - 主动挖掘客户需求(扩张销售线索);

2. **续约与增购**:
   - 续约提醒 + 续约谈判;
   - 推动客户内部"用起来"(提升使用率);
   - 转介绍激励(老客户带新客户);

3. **客户赋能**:
   - 每月组织"用户社区活动"(线上/线下);
   - 输出"客户成功案例"(可作为销售物料);
   - 推动客户参与"产品共建"(需求收集);

4. **数据监控**:
   - 监控客户健康度(登录率 / 模块使用率 / 流程时效);
   - 提前预警"沉默客户"(使用率下降);
   - 输出客户健康度报告(每月)。

**【任职要求】**

- 3-5 年 B2B 客户成功 / 大客户销售经验;
- 有 SaaS / 软件公司 CSM 经验优先;
- 优秀的客户沟通能力(尤其面对高层);
- **加分**:有制造业 IT 客户成功经验。

**【薪酬范围】** ¥12K-22K/月(13 薪,含绩效提成)

## 职能组

> 初期职能组大部分**兼职/外包**,Phase 2-3 再扩。

---

## 🟡 P1-01:HR(兼职/外包,0.5 人)

**【岗位职责】**

1. **招聘**:
   - 维护招聘渠道(Boss 直聘 / 猎聘 / 脉脉);
   - 筛选简历 / 约面试 / 跟进 offer;
   - 维护招聘漏斗看板(飞书/钉钉)。

2. **入职/离职**:
   - 办理入职手续(合同/社保/公积金);
   - 办理离职手续(交接/证明);
   - 维护员工档案。

3. **薪酬/绩效**:
   - 每月算工资(配合财务);
   - 季度绩效数据汇总;
   - 每年 4 月调薪数据。

4. **文化建设**:
   - 协助组织团队活动;
   - 维护内部沟通渠道(飞书/钉钉);
   - 员工关怀(节日福利、生日礼物)。

**【任职要求】**

- 3-5 年 HR 经验(优先有 IT/互联网公司经验);
- 熟悉劳动法;
- 熟悉社保/公积金;
- 良好的沟通能力;
- **合作模式**:初期兼职/外包(每月 8-16 小时),Phase 2 转全职。

**【薪酬范围】** ¥6K-10K/月(兼职顾问费) / ¥10K-15K/月(全职)

---

## 🟢 P2-01:行政(兼职/外包,0.5 人)

**【岗位职责】**

1. **办公场地**:租赁对接、物业、办公用品;
2. **差旅/报销**:差旅审批、机票/酒店、报销;
3. **证照**:公司证照、印章、档案;
4. **会议**:会议室预定、议程跟进;
5. **后勤**:员工活动、节日福利。

**【任职要求】**

- 1-3 年行政经验;
- 细心、有责任心;
- 熟练使用办公软件。

**【薪酬范围】** ¥5K-8K/月(兼职/全职)

## 🔴 P0-02:法务顾问(**关键岗位,外包/兼职**)

> **为什么是关键岗位**:B2B 制造业涉及合同、招投标、合规、知识产权,**没有法务会卡销售**。
> 初期可外包/兼职,M3 之后视业务量决定是否转全职。

**【岗位职责】**

1. **合同审核**:
   - 销售合同(主合同、SLA、保密协议);
   - 采购合同(供应商、外包);
   - 合作协议(联合开发、渠道合作);
   - 劳动合同(配合 HR)。

2. **合规**:
   - 等保 2.0 合规对接;
   - 《数据安全法》《个人信息保护法》合规评估;
   - 招投标法(制造业销售常涉);
   - 行业合规(ISO 9001 / IATF 16949 合同条款)。

3. **纠纷处理**:
   - 客户纠纷(质量问题 / 交付争议);
   - 员工纠纷(仲裁/诉讼);
   - 知识产权(代码/文档/商标/专利)。

4. **证照**:
   - 公司证照(营业执照、ICP、EDI);
   - 知识产权登记(软著、商标、专利);
   - 资质认证(高新企业、ISO 等,Phase 2 备用)。

5. **法律咨询**:
   - CEO 日常咨询;
   - 产品/运营/销售咨询;
   - 紧急响应(2 小时内)。

**【任职要求】**

- 5+ 年互联网/制造业法务经验;
- 熟悉《合同法》《公司法》《劳动法》《数据安全法》《个人信息保护法》《招投标法》;
- 持律师执业证;
- **加分**:
  - 有制造业 / 工业软件法务经验;
  - 有 ISO 9001 / IATF 16949 合同经验;
  - 有 DPO(数据保护官)认证。

**【合作模式】**

- 月度顾问费 + 按项目计费(合同审核、纠纷处理、证照办理);
- 月度 review 一次 + 应急随时响应(2 小时内);
- 关键合同(主合同)必须审核。

**【薪酬范围】** ¥8K-15K/月(顾问费) + ¥500-2000/项目

---

## 🟡 P1-02:财务(兼职/外包,0.5 人)

**【岗位职责】**

1. **记账**:
   - 日常记账(收入/支出);
   - 银行对账;
   - 应收/应付管理。

2. **税务**:
   - 月度报税;
   - 发票管理;
   - 税务筹划(Phase 2 备用)。

3. **薪酬**:
   - 每月算工资(配合 HR);
   - 社保/公积金代扣代缴。

4. **审计**:
   - 配合年度审计;
   - 配合税务稽查;
   - 配合政府补贴申报(高新企业等)。

**【任职要求】**

- 3-5 年财务经验(优先有 IT/软件公司经验);
- 持会计证;
- 熟悉小规模纳税人/一般纳税人税务;
- 熟练使用财务软件(金蝶/用友/柠檬云)。

**【合作模式】**:初期兼职/外包(每月 8-16 小时),Phase 2 转全职或继续外包。

**【薪酬范围】** ¥5K-8K/月(兼职顾问费) / ¥8K-12K/月(全职)

## 附录:实施组 + 职能组的"反模式"

| 反模式 | 表现 | 提问方式 |
|---|---|---|
| 实施只做"安装" | 不会做"业务配置" | "你在客户现场会做哪些事?" |
| 实施不会写文档 | "我口述" | "你的实施文档最长写过多少页?" |
| CSM 不会算账 | 不懂客户健康度 | "你怎么判断一个客户要流失了?" |
| HR 不会算工资 | 只会招聘 | "你之前每月自己算过工资吗?" |
| 法务不会审合同 | "模板给你了" | "上次审合同你发现了什么坑?" |
| 财务不会报税 | "找代账公司" | "你之前独立报过税吗?" |
