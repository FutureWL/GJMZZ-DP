# 03 三层组织与角色

> 来源:`design/Org/01_组织格局矩阵与角色映射.md`

## 1. 三层(横向)

| 层级 | 信息偏好 | 典型页面形态 | 常见交互 |
|---|---|---|---|
| **高层** | 摘要、趋势、异常、风险 | 驾驶舱/大数字/KPI 卡 | 下钻到域/部门/工厂;审批 |
| **中层** | 对标、结构、闭环、资源 | 部门看板 + 列表明细 | 分配任务、推动闭环、看超时 |
| **基层** | 任务、步骤、操作 | 列表/详情/表单/工单 | 处理、记录、提交、留痕 |

## 2. 角色样例(每层 5 个)

### 2.1 高层(5)
1. 总经理
2. 常务副总经理
3. 营销副总(分管市场/销售)
4. 制造副总(分管生产/质量/设备)
5. 财务/风控负责人(分管财务/审计/合规/内控)

### 2.2 中层(5)
1. 生产经理(制造部经理)
2. 质量经理
3. 采购经理
4. 设备/维护经理
5. 信息化/IT 经理(含数据安全接口人)

### 2.3 基层(5)
1. 班组长
2. 计划员/调度员
3. 检验员/质量工程助理
4. 维修技师
5. 物流/库管操作员

## 3. 角色 → 门户 → 默认落地页(模板)

> 同一个人可能属于多个域,默认落地页以"主要职责域"为主,其余域通过门户切换进入。

| 角色 | 主要域 | 默认落地页 | 常用路径 |
|---|---|---|---|
| 总经理 | 经营/管理 | 经营驾驶舱(集团/工厂切换) | 驾驶舱 → 异常摘要 → 下钻部门/工厂 → 审批中心 |
| 营销副总 | 经营 | 经营驾驶舱(营销) | 驾驶舱 → 订单/回款趋势(占位) → 大额审批(占位) |
| 制造副总 | 生产/管理 | 生产总览(工厂级) | 生产总览 → 告警中心 → 下钻产线 → 处置闭环 |
| 财务/风控负责人 | 管理/支持 | 管理驾驶舱(风险/超时) | 风险/超时 → 审批中心 → 审计整改(占位) |
| 生产经理 | 生产 | 产线/车间看板 | 看板 → 工单列表 → 工单详情 → 异常/停线 |
| 质量经理 | 生产/管理 | 质量看板 | 质量看板 → SPC → 追溯 → 整改闭环(占位) |
| 采购经理 | 管理/经营 | 采购驾驶舱/待办 | PR 列表 → 发起审批 → RFQ 比价 → 定标 → PO |
| 设备经理 | 生产 | 设备监控/OEE | 设备监控 → 告警 → 维修工单 → 完工留痕 |
| IT 经理 | 支持 | 支持门户首页(服务台) | IT 工单 → 受理 → 处理 → 验收 → 关闭 |
| 班组长 | 生产 | 班组任务与告警 | 告警 → 确认 → 处置留痕 → 关闭 |
| 计划员 | 生产/管理 | 排程(甘特) | 排程 → 工单详情 → 交期风险提示(占位) |
| 检验员 | 生产 | 检验任务列表 | 任务列表 → 记录详情 → 判定 → 附件留痕 |
| 维修技师 | 生产/支持 | 维修任务(移动/PC) | 任务 → 到场 → 完工 → 用料/工时留痕 |
| 物流/库管 | 生产/支持 | 物流台账(占位) | 入库/出库(占位) → 关联工单/批次(占位) |
