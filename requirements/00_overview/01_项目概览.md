# 01 项目概览

## 1. 一句话定位

**为高精密制造企业打造的"五域+三层"数字化平台**,把经营、管理、生产、支持和附加(党群/工会/妇联等)统一到一个门户里,让高层看驾驶舱、中层看闭环、基层看任务。

## 2. 仓库内名字

- 仓库名:`GJMZZ-DP`
- 平台标题:`高精密制造数字化平台`(见 `apps/portal-ui/src/app/appMeta.ts`)
- 命名空间 npm:`@factory/*`(见 `packages/mock-data/package.json`)

## 3. 目标与边界

### 3.1 目标(本期 MVP)
1. **统一入口**:一套 SSO(Keycloak)、一套门户切换、五个域的导航。
2. **可点击的高保真原型**:覆盖 P0 页面(见 `design/IA/02_页面清单与优先级.md`),能形成可演示流程。
3. **本地一键启动**:`docker compose up -d --build` 启动全栈(包含 Keycloak、Flowable、Postgres、MinIO、TDengine、CloudBeaver 等)。
4. **真后端雏形**:`factory-api`(NestJS + Prisma + Postgres)对核心域(审批中心/事故/工厂/菜单)有持久化实现,其余域用 mock-data 包占位。

### 3.2 边界(本期不做)
- 多租户(单租户集团)
- 真实生产数据接入(OT/SCADA 集成)
- 移动端 iOS 端(只做 Android Capacitor)
- 微服务拆分(单 NestJS monolith + 多镜像前端)
- 完整权限模型(只做 Keycloak realm + role,业务菜单权限暂为视觉位)

## 5. 用户角色(详见 `03_三层组织与角色.md`)

- **高层 5 人**:总经理、常务副总、营销副总、制造副总、财务/风控负责人
- **中层 5 人**:生产/质量/采购/设备/IT 经理
- **基层 5 人**:班组长、计划员、检验员、维修技师、库管员
