# 04 图表与数据可视化规范

## 1. 总原则（制造数据）
- 单位与口径优先：每张图都必须明确单位、统计口径、时间范围
- 阈值与目标优先：OEE/良率/节拍等建议提供目标线、上/下限带
- 告警优先：图表中的异常点要能对应到告警/事件（高亮与跳转位）
- 可读优先：减少装饰，提升对比与标注清晰度

## 2. 图表类型与适用场景
| 类型 | 适用场景 | 关键规范 |
|---|---|---|
| 折线/面积 | 趋势（产量、良率、温度、振动） | 异常点高亮；必要时使用移动平均 |
| 柱状/条形 | 对比（产线对比、TopN 停机原因） | 统一排序规则；TopN 保留“其他” |
| 堆叠柱 | 构成（停机时长构成、缺陷类型构成） | 堆叠段数建议 ≤ 6 |
| 甘特 | 排程、工序计划与实际 | 计划/实际用不同纹理或描边区分 |
| 玫瑰/饼图 | 仅用于占比很少分类且需要“结构感” | 不推荐大规模使用；优先条形 |
| 热力图 | 时段密度（告警热区、节拍热区） | 色阶必须有图例与阈值解释 |
| 散点 | 质量相关性（工艺参数 vs 缺陷） | 支持刷选与阈值线位 |
| 仪表/环形 | 单指标达成（OEE、良率） | 仅用于 1–2 个关键指标，避免满屏仪表 |

## 3. 配色策略
### 3.1 序列色（Categorical）
用于“多产线/多设备/多原因”对比。建议 8 色以内循环，超过则引入分组或交互筛选。

浅色建议（示例）
- C1 #2F6BFF（蓝）
- C2 #22C55E（绿）
- C3 #F59E0B（橙）
- C4 #EF4444（红）
- C5 #06B6D4（青）
- C6 #8B5CF6（紫）
- C7 #64748B（灰）
- C8 #F97316（橘）

深色主题：同色系提升亮度与饱和，避免在深背景上发灰；同时控制线宽与点大小，避免刺眼。

### 3.2 连续色（Sequential）
用于热力图/强度映射。建议单向色阶（浅→深）或双向色阶（低→中→高）。

建议
- 低值：bg-surface
- 中值：primary（降低饱和）
- 高值：warning/error（仅在确有业务含义时使用）

### 3.3 状态色（Status）
制造状态必须统一使用 Design Tokens 中的状态语义色（运行/停机/故障/离线/维护/告警）。

### 3.4 流程与风险色（Decision）
- 流程状态：使用 `flow-*`（草稿/审批中/通过/驳回/取消/超时）
- 风险等级：使用 `risk-*`（低/中/高/严重）
- 合规提醒：使用 `compliance-*`（证照到期/黑名单/违规）

## 4. 标注与坐标规范
- 标题：说明“指标 + 维度 + 时间范围”，例如“产线A 近7天 OEE（%）”
- 单位：放在轴标题或副标题，不要只写在 tooltip
- 数值格式：
  - 百分比：1 位小数（如 98.6%）
  - 时长：统一单位（min/h）并提供换算规则
  - 数量：千分位分隔（12,345）
- 网格线：弱化；突出阈值线/目标线
- Tooltip：必须包含完整维度信息（时间点、产线、设备、口径）

## 5. 告警与异常表达
- 异常点：红色描边 + 点高亮 + 旁注（可选）
- 阈值越界：背景带（band）提示安全区/风险区
- 事件轴：在趋势图下方可增加事件条（停机/换型/保养）

## 6. 图表密度与布局（后台 vs 大屏）
- PC：一屏 2–6 张图为宜，保留筛选与表格明细
- 大屏：优先大数字与少图；趋势图减少网格与小文本；告警区永远可见

## 7. 典型看板模板（建议）
- 工厂总览：KPI（产量/良率/OEE/在制）+ 产线矩阵 + 告警墙 + 今日Top异常
- 产线看板：节拍趋势 + 停机原因TopN + OEE 分解 + 关键设备状态
- 质量看板：良率趋势 + 缺陷TopN + SPC 摘要 + 追溯入口
- 设备看板：运行状态分布 + MTBF/MTTR 摘要 + 维修工单状态

## 8. 五域常用 KPI 卡模板（高/中/基层）
KPI 卡建议统一结构：标题（含口径位）+ 主数值 + 同比/环比 + 目标差距 + 异常提示（可选）+ 下钻入口。

### 8.1 经营域（Business）
- 收入/订单额/回款（币种与税制占位）
- 毛利率/贡献（口径占位）
- 订单交付风险（占位：按工厂/客户分布）

### 8.2 管理域（Management）
- 目标达成率（KPI）
- 预算执行率（预算）
- 流程超时率（审批/整改）
- 整改闭环率（审计/体系/EHS）

### 8.3 生产域（Production）
- 产量/达成率
- 良率/PPM
- OEE（含分解）
- 严重告警数/未关闭告警数

### 8.4 支持域（Support）
- 服务请求量与 SLA（IT/行政/支持类）
- IT 工单响应/解决时长（占位）
- EHS 隐患未闭环数（占位）
- 数据访问异常/高风险事件数（占位）

### 8.5 附加域（Additional）
- 培训覆盖率/参与率（占位）
- 活动触达率（占位）

## 9. 管理域供应链与人才常用指标图（采购/供应商/外协/招聘）
### 8.1 采购分析（Spend）
- 花费结构（品类/项目/供应商）：条形或堆叠条形；默认 TopN + 其他
- 价格趋势（关键物料）：折线；必须包含币种与税含/未含口径位
- 节省（目标价 vs 成交价）：对比柱或瀑布图（可选）
- 交付及时率（OTD）：折线 + 阈值线（目标线）

### 8.2 供应商（SRM）
- 准时交付率（OTD）：折线/条形对比（供应商 TopN）
- 来料质量（PPM/不良率）：折线 + 阈值线；异常点可关联到批次/检验
- 风险事件趋势：折线 + 事件标记（黑名单/违规/重大投诉）
- 合规到期分布：条形（按证照类型/到期月份）

### 8.3 外协工厂
- 产能负荷：热力图或堆叠条形（计划 vs 已占用 vs 可用）
- 交期达成：条形（工厂对比）+ 趋势（按周/月）
- 外协质量：TopN 缺陷/返工率（条形）+ 趋势（折线）

### 8.4 人才（招聘与编制）
- 编制与在岗：对比柱（编制/在岗/缺口）
- 招聘漏斗：漏斗图或分段条（投递/初筛/面试/Offer/入职）
- 岗位周期：箱线图（可选）或条形（平均招聘周期）

## 10. 强制“口径位”清单（经营/管理/支持类图表）
- 组织范围：集团/工厂/部门/产线（根据页面层级显示）
- 时间范围：日/周/月/季度（明确起止）
- 币种与税制：币种、税含/未含
- 数据来源：ERP/SRM/MES/HR/财务/ITSM/安保/EHS/数据安全（占位即可，但必须展示）
