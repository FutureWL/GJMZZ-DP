# 02 页面清单与优先级（P0/P1）

## 1. 定义
- P0：必须出高保真并可形成可点击流转（第一轮评审与汇报用）
- P1：需要出高保真静态稿（第二轮补齐，或作为扩展）
- P2：仅需模板/线框定义（后续按需补齐）

## 2. PC Web（五域门户）
### 2.1 全局通用（P0）
- 登录页（浅/深两套）
- 门户切换（经营/管理/生产/支持/附加）
- 组织范围选择（集团/工厂/部门/产线：按门户显示适当粒度）
- 全局搜索（结果按五域分组）
- 消息/待办中心（告警 + 待办）
- 审批中心（待办/已办/抄送）
- 审批详情（流程节点、意见、附件位）
- 用户入口（个人信息/退出：视觉位即可）

### 2.2 经营门户（Business）
- P0：经营驾驶舱首页（摘要：收入/订单/回款/毛利占位 + 风险提示）
- P1：经营分析（按工厂/部门/客户/产品/时间切片，占位）
- P1：营销/市场/销售模块入口页（占位）

### 2.3 管理门户（Management）
#### 2.3.1 计划与运营管理（占位为主）
- P1：目标与 KPI（达成率看板，占位）
- P1：预算与经营计划（预算执行看板，占位）
- P1：运营分析（占位）

#### 2.3.2 流程与合规管理
- P1：制度/流程库（占位）
- P1：内控/审计整改（占位）

#### 2.3.3 采购管理
- P0：采购申请 PR 列表
- P0：采购申请 PR 详情（含审批轨迹位）
- P0：询价/比价 RFQ 列表
- P0：RFQ 详情（供应商报价对比表）
- P0：定标/中标结果确认页（视觉位）
- P0：采购订单 PO 列表
- P0：采购订单 PO 详情（交付/收货节点位）
- P1：收货/对账/发票台账（仅视觉位）
- P0：采购分析看板（花费结构、价格趋势、节省、交付及时率）

#### 2.3.4 供应商管理（SRM）
- P0：供应商列表（标签：资质/风险/绩效）
- P0：供应商详情（概览 + 资质证照 + 交付与质量 + 风险事件）
- P0：供应商准入申请（表单 + 附件位）+ 审批轨迹位
- P1：绩效评分配置（规则位）

#### 2.3.5 外协工厂管理
- P0：外协工厂列表（产能/准时率/质量）
- P0：外协工厂详情（产能日历位、在制/交付、质量摘要）
- P0：外协工单列表
- P0：外协工单详情（计划/进度/质检节点位）
- P1：对账结算（仅视觉位）

### 2.4 生产门户（Production）
#### 2.4.1 总览
- P0：首页/总览（工厂级）
- P0：告警中心（列表 + 详情抽屉）

#### 2.4.2 生产与排程
- P0：工单列表（筛选/批量操作视觉位）
- P0：排程页（甘特：产线/工序维度切换）
- P0：工单详情（进度、工序、报工、异常、关联质量/设备）
- P1：订单列表/订单详情
- P1：异常列表/停线记录详情

#### 2.4.3 质量与追溯
- P0：检验任务列表
- P0：检验记录详情（结果、图片/附件位、判定）
- P0：SPC 看板（趋势/控制图 + 规则触发提示位）
- P0：追溯查询（输入/扫码位）+ 追溯链路详情
- P1：不良/缺陷分析（Pareto/分布/TopN）

#### 2.4.4 设备与维护
- P0：设备台账（列表/筛选）
- P0：设备监控（状态 + 关键点位 + 事件时间轴位）
- P0：OEE 看板（OEE/Availability/Performance/Quality）
- P0：维修工单列表 + 工单详情（派工/进度/用料位）
- P1：点检/保养计划与任务
- P2：备件与耗材

  - P0：体系文件库（占位）
  - P0：内审/外审问题台账（占位）
- 安保（P0/P1）
  - P0：安保事件台账（占位）
  - P1：巡检/巡更任务（占位）
- 信息/IT（P0）

## 3. 大屏（可视化）
- P0：工厂总览（KPI + 产线矩阵 + 告警墙）
- P0：产线总览（节拍/产量/良率/OEE + 异常）
- P0：告警墙（严重度分栏 + 最新告警滚动）
- P1：质量大屏（良率/缺陷 TopN/SPC 摘要）
- P1：设备大屏（停机原因分布、MTBF/MTTR 摘要）

## 4. 移动端 H5
- P0：告警列表（筛选：工厂/产线/设备/严重度）
- P0：告警详情（确认/转派/关闭，处置留痕位）
- P0：任务列表（点检/维修/检验统一入口或分 Tab）
- P0：工单处理（接单→到场→完工）
- P0：扫码入口（追溯/设备/工单）
- P1：看板首页（KPI 卡 + 我关注）
- P1：我的（班次/值班、消息、设置）

## 5. P0 可演示的端到端流转
1) 告警处置闭环：告警列表 → 告警详情 → 确认/转派/关闭 → 留痕
2) 维修工单闭环：工单列表 → 工单详情 → 状态推进（移动端可补充“到场/完工”）
3) 追溯查询：全局搜索/扫码 → 追溯结果 → 链路详情（关联工单/设备/检验）
4) 采购闭环：PR 列表 → PR 详情 → 发起审批 → RFQ/比价详情 → 定标确认 → PO 详情（收货节点位）
5) 供应商准入：供应商列表 → 准入申请 → 审批中心 → 供应商详情（证照/风险/绩效）
6) 招聘编制：编制总览 → 招聘需求申请 → 审批中心 → 岗位/候选人漏斗
7) IT 工单闭环：IT 工单列表 → 工单详情 → 受理/处理 → 验收 → 关闭（占位）
