# 01 站点地图与导航（PC / 大屏 / 移动）

## 1. 总体信息架构（按五域）
### 1.1 经营域（Business Portal）
- 经营总览
  - 经营驾驶舱（摘要）
  - 经营分析（可选）
- 营销/市场/销售（本期可先占位）
  - 客户与机会（占位）
  - 订单与合同（占位）
  - 回款与预测（占位）

### 1.2 管理域（Management Portal）
- 计划与运营管理
  - 目标与 KPI（占位）
  - 预算与经营计划（占位）
  - 运营分析（占位）
- 流程与合规管理
  - 审批中心（待办/已办/抄送）
  - 制度/流程库（占位）
  - 内控/审计整改（占位）
- 供应链管理（管理视角）
  - 采购管理（PR/RFQ/PO/分析）
  - 供应商管理（准入/绩效/风险合规）
  - 外协工厂管理（外协工单/产能交期/质量）

### 1.3 生产域（Production Portal）
- 总览
  - 工厂总览
  - 产线总览
  - 今日焦点（产能/交付/告警/质量）
- 生产与排程
  - 工单/批次
  - 排程（甘特/日历）
  - 报工与进度
  - 异常与停线记录
- 质量与追溯
  - 检验任务
  - SPC（趋势/控制图）
  - 不良/缺陷分析
  - 追溯（批次/序列）
- 设备与维护
  - 设备台账
  - 设备监控（状态/OEE）
  - 点检/保养
  - 维修工单
- 告警与消息
  - 告警中心（统一入口）
  - 消息中心
  - 值班与处置（可选：岗位/班次）

### 1.4 支持域（Support Portal）
- 支持门户首页（服务台/待办/风险提示）
- 行政（占位）
- 人事（组织/编制/招聘/培训占位）
- 财务（费用/预算占位）
- 审计（计划/整改占位）
- 体系（文件库/内审占位）
- 安保（事件/巡检占位）
- 信息/IT（IT 工单/资产台账占位）
- 数据安全（分级/访问审计占位）
- 安全环保 EHS（隐患/整改闭环占位）

### 1.5 附加域（Additional Portal）
- 人才发展中心（占位）
- 党群（占位）
- 工会（占位）
- 妇联（占位）

### 1.6 系统与配置（全局）
- 组织/工厂/产线/部门
- 用户/角色/权限（视觉层级即可）
- 字典/标签/规则（可选）

## 2. PC Web 导航模型（推荐）
### 2.1 基础布局
- 左侧侧边栏（一级模块）
- 顶部全局区（门户切换、组织范围选择、时间范围、全局搜索、消息/待办、用户入口）
- 内容区：页面标题区 + 筛选区 + 主内容（图表/表格/详情）

### 2.2 关键导航与全局控件
- 门户切换
  - 入口：顶部左侧（工厂选择器旁）或用户菜单
  - 选项：经营门户 / 管理门户 / 生产门户 / 支持门户 / 附加门户
  - 行为：切换后保持组织范围/时间范围；清空业务筛选（避免口径混用）
- 组织范围选择器
  - 位置：顶部左侧
  - 范围：集团/工厂/部门/产线（根据当前门户显示适当粒度）
- 时间范围选择器
  - 默认：今日/近 8 小时（大屏），近 7 天（办公端可切换）
  - 快捷：今日/本周/本月/自定义区间
- 全局搜索
  - 目标：工单/设备/批次/序列/告警/PR/PO/供应商/外协工厂/人员（占位）
  - 结果：按“五域”分组；点击条目自动切换到对应门户并打开详情
- 消息/待办（永远可见）
  - 组成：告警（生产域）+ 待办（管理/支持域）
  - 行为：点击进入统一中心（列表 + 详情抽屉/页）

### 2.3 跨域跳转规则（推荐）
- 审批中心：从任意门户进入，保持当前门户不强制切换
- 全局搜索：结果按域分组；进入详情时自动切换门户
- 详情互链：在详情页中用“关联对象卡片”跳转（例如：PO 关联供应商、外协工单关联工单/批次）

## 3. 大屏导航模型（推荐）
### 3.1 大屏类型
- 工厂总览屏：多产线对比 + 告警墙 + 关键 KPI
- 产线/工序屏：单产线深挖（节拍/良率/OEE/瓶颈）
- 告警墙：按严重度、时间、区域聚合

### 3.2 大屏交互（最小化）
- 轮播策略：按“工厂总览 → 产线A → 产线B → 告警墙”轮播
- 手动切换：右上角模式切换按钮（可隐藏）
- Drill-down：点击产线卡片进入产线屏（可选）

## 4. 移动端导航模型（推荐）
### 4.1 底部 Tab（四栏）
- 看板：简化 KPI + 趋势 + 我关注的设备/工单
- 告警：告警列表/筛选/处置
- 任务：点检/保养/维修工单/检验任务
- 我的：班次/值班、消息、设置

### 4.2 移动端全局能力
- 扫码入口：追溯/设备/工单（置于告警或任务页的浮动按钮）
- 消息推送入口：未读与告警摘要
